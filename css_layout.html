Box model PE

Usamos el box model para editar el borde de las imagenes, frases,... Se usan: width, height, padding, border y margin

-Width y height se utilizan para cambiar el ancho y el alto de la imagen
-Padding lo usamos para elegir cuanta separación queremos del margen
-Border se usa para cambiar el borde de la imagen
-Margin se usa para elegir cuanta distancia queremos que haya entre los elementos
