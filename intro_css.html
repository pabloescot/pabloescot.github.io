Selección por etiqueta PE

Se puede cambiar el fondo o el color de las letras de un mismo tamaño en style, por ejemplo: h3{ color: "color elegido" }

Este texto está en tamaño h3

Selección por id PE

Se puede elegir el color del fondo o de las letras de un apartado usando id, por ejemplo: en el apartado que quieras colorear se usa detras del tamaño de letra id="nombre", luego en style hay que usar #"nombre"{color: "color seleccionado"}

Selección por class PE

Se usa de la misma forma que id, pero este sirve para seleccionar varias líneas, pero con este hay que usar un punto delante del nombre, por ejemplo: ."nombre"

He cambiado el tipo y el tamaño de letra en el apartado de style dentro de body con las siguientes funciones: font-family:"tipo de letra"; (para el tipo de letra) y font-size:"Xpx"; (para el tamaño)
